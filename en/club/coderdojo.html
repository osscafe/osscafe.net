CoderDojo Tokyo @下北沢

CoderDojo is a volunteer-run programming dojo for kids!
CoderDojo Tokyo meets every Sunday.

Let's try programming!

  • Scratch (program with visual blocks)
  • HTML / CSS / JavaScript
  • Make your own website using tools like YouTube
  • Program with MAKE Kits and other hardware

...and much more, depending on interest!

Mentors & Staff

Mentors can be programmers, designers, engineers, or anyone with an interest in software or hardware. For full details see CoderDojo Tokyo!
